# Sistem-Automatizat-De-Management-Si-Distributia-Lichidelor

Acest proiect reprezintă o implementare hardware și software pentru un sistem de distribuție automatizată a lichidelor, utilizând un microcontroller Arduino. Sistemul permite utilizatorului să selecteze băuturi și cantități dintr-un meniu afișat pe un ecran OLED, iar distribuția este realizată prin intermediul unor pompe controlate cu un driver L298N. Proiectul integrează mai multe componente hardware și software, inclusiv senzori precum cel ultrasonic, joystick-uri, și funcții de meniu interactiv.
